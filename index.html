School ERP

I’ve built a complete School ERP system using Vue.js, Tailwind CSS, and AG Grid. It’s a fully responsive, single-page dashboard that allows admins to manage student records efficiently.

The system includes features like dynamic data tables with filtering, sorting, and action buttons for editing and deleting entries. I integrated backend APIs to fetch real-time student data, including nested details like guardians and academics, and made sure the UI feels clean and intuitive.

The layout includes a sidebar, top navigation bar, and modular pages for registration, student management, and more. It’s designed to be scalable, fast, and ready for future enhancements like role-based access, analytics, or export features.
